## Overview

LocaLore is an interactive, collaborative platform for documenting and exploring folklore creatures from around the world. Built with modern web technologies, it provides an intuitive map-based interface for discovering mythical beings, legendary monsters, and cultural legends from different regions.

## ✨ Features

- 🗺️ **Interactive Global Map** - Explore creatures by geographical location
- 📸 **Image Gallery** - Browse and upload creature imagery
- 🌍 **Crowdsourced Data** - Community-driven folklore database
- 🔐 **Secure Backend** - Row-level security with Supabase
- ⚡ **Fast Performance** - Built with Vite for rapid development and optimized builds
- 🎨 **Modern UI** - Responsive design with Tailwind CSS
- 📱 **Mobile-Friendly** - Works seamlessly across devices

## 🛠️ Tech Stack

- **Frontend**: React with TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Mapping**: Leaflet
- **Routing**: React Router
- **Backend/Database**: Supabase (PostgreSQL)
- **Storage**: Supabase Storage

**Language Composition**:
- TypeScript: 92.9%
